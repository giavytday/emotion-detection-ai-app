Emotion Detection AI Application
A professional AI-driven web application that leverages IBM Watson NLP libraries to perform real-time emotion analysis on text data. This project was developed as part of the "AI Web Application Development" course.

🚀 Project Overview
This application allows users to input statements (such as customer feedback) and receive a detailed breakdown of the emotions conveyed. Unlike basic sentiment analysis (positive/negative), this system extracts scores for five specific emotions: Joy, Anger, Fear, Sadness, and Disgust, and identifies the Dominant Emotion.

✨ Key Features
Deep Emotion Analysis: Uses pre-trained Watson NLP models for high-accuracy text analysis.
Web Interface: A clean, user-friendly front-end built with HTML/CSS and Flask.
Robust Error Handling: Specifically handles empty or invalid inputs to prevent system crashes.
Professional Standards: Code is PEP8 compliant, achieving a perfect 10/10 score on Pylint static analysis.
Unit Tested: Includes a suite of automated tests to ensure the logic remains accurate.

🛠️ Tech Stack
Language: Python 3
Framework: Flask (Web Server)
AI Engine: IBM Watson NLP via Embeddable AI libraries
Testing: Unittest (Python's built-in testing framework)
Linter: Pylint (Static Code Analysis)
